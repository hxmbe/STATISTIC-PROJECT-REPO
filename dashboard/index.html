Key Findings

Doral Academy students sleep an average of 6.42 hours per night — statistically significantly less than the CDC-recommended 8 hours (t = −14.48, p ≈ 0.000). However, the relationship between sleep and unweighted GPA is not statistically significant (r = −0.011, p = 0.917). Sleep hours alone do not linearly predict GPA in this sample, though students are substantially sleep-deprived relative to national benchmarks.

Python Analysis Code (Extra Credit)

Full annotated hand-calculation verification — all formulas derived from scratch, then confirmed with scipy.

analysis/sleep_gpa_analysis.py 
# ============================================================
# AP Statistics Project — Sleep & GPA  |  Doral Academy 2026
# Extra Credit: Annotated Python verification of hand calcs
# ============================================================
import pandas as pd
import numpy as np
from scipy import stats

# ── Load & clean data ──────────────────────────────────────
df = pd.read_csv('Statistics_Research_.csv')
df.columns = ['timestamp','grade','gpa_unweighted','gpa_weighted','sleep_raw']

def parse_sleep(val):
    """Handle messy entries: '7 hours', '6-8', '4 or 5'"""
    val = str(val).strip().lower().replace('hours','').replace('hour','').replace('>','').strip()
    if '-' in val:
        p = val.split('-')
        try: return (float(p[0].strip()) + float(p[1].strip())) / 2
        except: return np.nan
    if 'or' in val:
        p = val.split('or')
        try: return (float(p[0].strip()) + float(p[1].strip())) / 2
        except: return np.nan
    try: return float(val)
    except: return np.nan

df['sleep'] = df['sleep_raw'].apply(parse_sleep)
df['gpa']   = pd.to_numeric(df['gpa_unweighted'], errors='coerce')
df = df.dropna(subset=['sleep','gpa'])
n, sleep, gpa = len(df), df['sleep'].values, df['gpa'].values

# ── Descriptive stats (hand) ───────────────────────────────
x_bar = np.sum(sleep) / n                          # x̄ = Σxᵢ / n
s     = np.sqrt(np.sum((sleep - x_bar)**2) / (n-1)) # s = √(Σ(xᵢ-x̄)²/(n-1))
SE    = s / np.sqrt(n)                               # SE = s/√n

# ── 95% CI for μ_sleep ────────────────────────────────────
t_star = stats.t.ppf(0.975, df=n-1)
CI = (x_bar - t_star*SE, x_bar + t_star*SE)
print(f"95% CI: ({CI[0]:.4f}, {CI[1]:.4f}) hours")

# ── One-sample t-test: H₀: μ = 8 ─────────────────────────
t_stat = (x_bar - 8.0) / SE
p_val  = 2 * stats.t.cdf(t_stat, df=n-1)
print(f"t = {t_stat:.4f},  p = {p_val:.8f}")

# ── Linear regression (hand) ───────────────────────────────
y_bar = np.mean(gpa)
Sxy   = np.sum((sleep - x_bar) * (gpa - y_bar))  # Σ(xᵢ-x̄)(yᵢ-ȳ)
Sxx   = np.sum((sleep - x_bar)**2)               # Σ(xᵢ-x̄)²
b1    = Sxy / Sxx                                   # slope
b0    = y_bar - b1 * x_bar                          # intercept
r     = Sxy / np.sqrt(Sxx * np.sum((gpa - y_bar)**2))  # correlation
print(f"ŷ = {b0:.4f} + ({b1:.4f})x   r = {r:.4f}")

# Verify with scipy ────────────────────────────────────────
slope_v, intercept_v, r_v, p_v, se_v = stats.linregress(sleep, gpa)
print(f"scipy: slope={slope_v:.4f}, r={r_v:.4f}, p={p_v:.4f}  ✓")
